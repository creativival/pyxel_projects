# Pixel Projects

Pixelのゲームを作成します。

## 環境

- Mac OS X 10.15.7
- MiniConda
- Python 3.10
- Pygame 2.1.8

## インストール

```bash
$ conda create -n pixel_projects python=3.10
$ conda activate pixel_projects
$ pip install pygame
```

# サンプルプロジェクト

/site-packages/pygame/examples/ にサンプルがあります。

root/examples/ にコピーして実行します。
